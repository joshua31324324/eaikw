## Typography

### Font Stack
```css
Primary Font: Inter (sans-serif)
Fallback: -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif
Font Weights: 400 (Regular), 500 (Medium), 600 (Semibold), 700 (Bold)
```

**Why Inter?** Clean, modern, highly legible at all sizes, excellent for web.

### Type Scale

| Element | Size | Weight | Line Height | Letter Spacing | Usage |
|---------|------|--------|-------------|-------------------|-------|
| **H1** | 48px | 700 | 1.2 | -0.03em | Page headlines, hero sections |
| **H2** | 36px | 700 | 1.2 | -0.02em | Section headings |
| **H3** | 28px | 600 | 1.3 | -0.01em | Subsection headings |
| **H4** | 20px | 600 | 1.4 | 0 | Card titles, small headings |
| **Body** | 16px | 400 | 1.6 | 0 | Primary body text |
| **Small** | 14px | 400 | 1.5 | 0 | Secondary text, captions |
| **Xs** | 12px | 500 | 1.4 | 0.1em | Labels, overlines |
| **Button** | 14px | 600 | 1.2 | 0.05em | All button text |

### Responsive Type
```css
H1: clamp(32px, 8vw, 48px)
H2: clamp(24px, 6vw, 36px)
H3: clamp(20px, 4vw, 28px)
Body: 16px (fixed)
```

### Usage Guidelines

**H1 - Headlines**
- Used once per page (main headline)
- Sets page context
- Example: "Build Your AI Career"

**H2 - Section Headings**
- Organize major content sections
- Example: "Four Pillars of Success"

**H3 - Subsections**
- Organize content within sections
- Example: "1. Guided Onboarding"

**H4 - Card/Component Titles**
- Used in cards, modals, components
- Example: "Month 1: Foundation & Planning"

**Body Text**
- Primary reading text
- 16px, line height 1.6 for readability
- Max line length: 600-750px for paragraphs

**Small Text**
- Supporting info, dates, metadata
- Not for critical information
- Example: "Updated 3 days ago"

## Writing Guidelines by Content Type

### Headlines
- **Action-oriented:** Use verbs (Build, Learn, Join, Create)
- **Benefit-focused:** Answer "Why should I care?"
- **Clear:** No jargon or clever wordplay
- **Length:** 3-8 words ideal
- **Case:** Title Case or Sentence case (not ALL CAPS)

### Body Copy
- **Scannable:** Short paragraphs (2-3 sentences)
- **Specific:** Numbers, examples, concrete language
- **Benefit-driven:** Start with "you" benefit
- **Action-oriented:** What should reader do?
- **Length:** 15-20 word sentences typical, vary for rhythm

### Calls-to-Action
- **Specific:** What happens next?
- **Verb-driven:** "Join", "Register", "Explore", "Download"
- **Benefit-focused:** "Join 200+ students" not just "Join"
- **Examples:**
  - ‚úÖ "Start My Personalized Path"
  - ‚úÖ "Explore the 6-Month Timeline"
  - ‚ùå "Click Here"
  - ‚ùå "Submit Form"

### Microcopy
- **Helpful:** Guides user action
- **Clear:** No ambiguity about what's happening
- **Brief:** One line or less
- **Examples:**
  - "This helps us personalize your path"
  - "Required for mentor matching"
  - "Event ends in 3 days"
